# SSC GK Learning App

**Version:** 0.1.0

**Status:** In Development

---

# Vision

Build a modern, offline-first learning platform for SSC, Railways, Banking, UPSC, and other competitive examinations.

The application should not only help users practice MCQs but also guide them through a complete learning cycle:

Read Notes → Practice Questions → Analyze Mistakes → Revise → Mock Test → Improve Performance

---

# Objectives

* Learn React, Node.js, Express, SQLite, and full-stack development.
* Create a high-quality portfolio project.
* Build an application that can be used daily for competitive exam preparation.
* Support multiple competitive examinations using the same software.
