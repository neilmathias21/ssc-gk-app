# SSC GK Learning App

A personal web application for preparing for SSC, Railway, Banking, and other competitive exams.

## Features

- Practice Mode (Random MCQs)
- Instant answer checking
- Detailed explanations
- Mock Tests (25 questions, 15-minute timer)
- Bookmarks
- Wrong Question Bank
- Statistics Dashboard
- Search Questions
- Course Notes

## Tech Stack

- React
- Node.js
- Express
- SQLite
- Git & GitHub
